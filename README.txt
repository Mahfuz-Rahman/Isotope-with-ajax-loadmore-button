## == Description ==

Isope custom post with ajax load more button. After clicking `Load More` button it'll append
more posts from WP -> DB

## == Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_shortcode('loader_posts_display'); ?>` in your templates

## == Shortcode Attributes ==
[loader_posts_display count="9" ]

### param 1 : count
